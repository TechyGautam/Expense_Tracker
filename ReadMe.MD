# 💰 Expense Tracker

A simple and interactive Expense Tracker built using Python, Streamlit, SQLite3, and Object-Oriented Programming (OOP). This application helps users manage their daily expenses, store data in a database, visualize spending patterns, and import expense records through CSV files.

## 🚀 Features

* ➕ Add new expenses
* 🗑️ Delete existing expenses
* 📋 View all expenses
* 📊 Expense Analytics

  * Bar Chart Visualization
  * Pie Chart Visualization
* 📂 Import expenses from CSV files
* 👀 View uploaded/imported data
* 💾 Persistent storage using SQLite3 database
* 🏗️ Built using Object-Oriented Programming principles

## 🛠️ Technologies Used

* Python
* Streamlit
* SQLite3
* Pandas
* Matplotlib
* Object-Oriented Programming (OOP)

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/TechyGautam/Expense_Tracker.git
cd Expense_Tracker
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```

## 📂 CSV Format

The uploaded CSV file should follow this format:

| Amount | Category | Date       | Description |
| ------ | -------- | ---------- | ----------- |
| 500    | Food     | 2026-06-18 | Pizza       |
| 1000   | Travel   | 2026-06-18 | Bus Ticket  |
